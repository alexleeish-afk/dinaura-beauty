# ✨ Dinaura França Beauty

Site institucional do **Studio Dinaura França Beauty**, especializado em estética facial, design de sobrancelhas, maquiagem e formação profissional.

🔗 **Acesse:** [dinaura.surge.sh](https://dinaura.surge.sh)

---

## 📋 Sobre o Projeto

Landing page moderna e responsiva desenvolvida para apresentar os serviços do studio, permitir agendamentos via WhatsApp e divulgar cursos de formação profissional na área da beleza.

---

## 🛠️ Tecnologias

- **HTML5** — Estrutura semântica
- **CSS3** — Estilização com variáveis CSS, gradientes, grid e flexbox
- **JavaScript** — Formulários com envio via WhatsApp, modais e scroll suave
- **Font Awesome** — Ícones
- **Surge.sh** — Hospedagem

---

---

## 🔹 Funcionalidades

- **Quem sou eu** — Apresentação da profissional
- **Serviços** — Catálogo completo de procedimentos estéticos
- **Cursos** — Formações com inscrição via modal e envio pelo WhatsApp
- **Agendamentos** — Formulário integrado com WhatsApp
- **Contatos** — WhatsApp, Instagram e endereço do studio

---

## 🚀 Como rodar localmente

Basta abrir o arquivo `index.html` no navegador ou usar um servidor local:

```bash
npx serve .
```

---

## 📦 Deploy

O site é publicado no Surge.sh:

```bash
surge . dinaura.surge.sh
```
